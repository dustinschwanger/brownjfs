# Brown County Department of Job & Family Services Website

Official website for the Brown County Department of Job and Family Services.

## Features

- 100% WCAG 2.2 AA Compliant
- Responsive design using USWDS (U.S. Web Design System)
- Accessible navigation and interactive elements
- Mobile-friendly chat widget
- Service information and resources

## Accessibility

This website has been built to meet WCAG 2.2 Level AA compliance standards, including:

- Semantic HTML5 structure
- ARIA landmarks and labels
- Keyboard navigation support
- Screen reader compatibility
- Color contrast ratios meeting AA standards
- Responsive text sizing
- Skip navigation links
- Focus indicators on all interactive elements
- Reduced motion support
- High contrast mode support

## Deployment

### Railway

This site is configured for deployment on Railway.

1. Connect your GitHub repository to Railway
2. Railway will automatically detect the Node.js project
3. The site will be deployed using the Express server

### Local Development

```bash
# Install dependencies
npm install

# Run the development server
npm start

# The site will be available at http://localhost:3000
```
